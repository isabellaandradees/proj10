# Explicação da Atividade

## Exercício 1
Neste exercício foi criado um vetor para armazenar 10 números digitados pelo usuário.  
Depois, utilizando uma estrutura de repetição e uma condição `if`, o programa verificou quais números eram pares e realizou a soma desses valores.

---

## Exercício 2
Neste exercício foi criado um vetor contendo 7 números inteiros.  
Em seguida, os valores armazenados no vetor foram exibidos no console utilizando `console.log()`.

---

## Exercício 3
Neste exercício o programa solicitou 10 números ao usuário e armazenou em um vetor.  
Depois, foi solicitado outro número para pesquisa.  
Utilizando uma estrutura de repetição e uma condição `if`, o programa contou quantas vezes o número apareceu no vetor.

---

## Exercício 4
Neste exercício o programa armazenou 10 números em um vetor e criou um novo vetor sem valores repetidos.  
Foi utilizada uma estrutura de repetição para verificar se o número já existia no novo vetor antes de adicioná-lo.
